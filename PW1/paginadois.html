A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que seignifica Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.
